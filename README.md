# Personal Info Form (Python)

A simple Python program that collects basic user information and
displays the data along with their data types and memory IDs.

This project is useful for beginners to understand: - User input in
Python - Data types (`str`, `int`, `float`) - Basic arithmetic
operations - The `type()` and `id()` functions

## 🚀 Features

-   Takes user input for:
    -   Name
    -   Age
    -   Height
    -   Favourite number
-   Displays:
    -   Value entered
    -   Data type of the value
    -   Memory ID of the variable
-   Calculates an estimated birth year using the entered age.

## 🛠️ Technologies Used
